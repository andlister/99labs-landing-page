Microsoft Stack — SMB Security Tiers

A clear progression from Basic → Silver → Gold. Short summaries first, full comparison below.

Prosumer — Solo / Micro

Freelancer / 1–5 devices. Self-managed.

  • Defender AV + OS auto-update
  • BitLocker / FileVault, OneDrive
  • MFA (Authenticator), password manager
  • DNS filter (optional)
$0–5 / user / mo

Basic — Secure Foundations

Prosumer / 1–10 users. Minimum viable controls.

  • Defender Antivirus (Windows/macOS)
  • BitLocker / FileVault, OneDrive versioning
  • MFA via Microsoft Authenticator
  • EOP email filtering baseline
  • 8x5 helpdesk
  • Onboarding / leavers
  • Patch mgmt
  • M365 backup
  • Monthly health report
MSP Sell: $35–45 / user / mo

Silver — Hardened & Managed

Growing SMB / 10–150 users. Centralized management.

  • Intune MDM + compliance
  • Defender for Business (EDR)
  • Conditional Access (Entra ID P1)
  • Defender for Office 365 Plan 1
  • 8x5 helpdesk + SLAs
  • Onboarding automation
  • Policy & CA tuning
  • Phishing sims & training
  • Asset mgmt • Quarterly vCIO
MSP Sell: $60–85 / user / mo

Gold — Full Defender XDR

Mature / regulated SMB. XDR, automation, compliance.

  • M365 Defender XDR + AIR
  • TVM / Exposure Mgmt
  • Defender for Office 365 Plan 2
  • Purview DLP • Sentinel/MDR (optional)
  • 24x7 or SOC/MDR
  • Compliance pack
  • IR playbooks
  • SIEM dashboards
  • Quarterly exercises
MSP Sell: $90–130 / user / mo
